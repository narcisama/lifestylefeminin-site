# ARHITECTURA - lifestylefeminin.ro

## Decizii tehnice cheie

### Stack
- **Astro 5** ales pentru: SSG zero-JS overhead, performance perfect,
  Core Web Vitals verde
- **Tailwind v4** ales pentru: utility-first, build size mic, theme via @theme
- **TypeScript strict** ales pentru: type safety, refactoring sigur

### Strategie SEO
- Schema.org Person + WebSite + sameAs declarare network
- llms.txt pentru AI Search (GPTBot, PerplexityBot, ClaudeBot)
- robots.txt permisiv pentru toate AI crawlers
- Canonical pe toate paginile
- OG + Twitter Cards complete

### Strategie partener network
- Site partener: sonia11.com
- Cross-link DOFOLLOW (NU sponsored, NU nofollow)
- Schema sameAs reciproca pe ambele site-uri
- Linkuri catre partener DOAR in: /despre/, author bio, footer
- ZERO mentionare OnlyFans pe lifestylefeminin.ro

## Deploy
- Target: Cloudflare Pages (gratis, fast, anti-DDoS)
- Domain: lifestylefeminin.ro (Hostico, DNS la Cloudflare)
- Branch deploy: main -> production
