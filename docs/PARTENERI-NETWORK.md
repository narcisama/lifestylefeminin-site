- **Locatii link:** /despre/, author bio articole, footer

## De ce DOFOLLOW (nu sponsored/nofollow)
Conform Google Spam Policies 2024-2026:
- `rel="sponsored"` = pentru linkuri PLATITE (afiliere comerciala) - NU
- `rel="nofollow"` = pierdere autoritate inutila - NU
- `rel="ugc"` = pentru comentarii utilizatori - NU
- **DOFOLLOW + Schema sameAs = pattern legitim pentru site-uri proprii** - DA

Acesta este pattern-ul folosit cu succes in vinderapid.ro <-> tuonline.ro.

## Reguli stricte network
1. NU duplicate content intre site-uri
2. MAX 1-2 linkuri cross-site per articol
3. NU anchor text manipulativ (ex: "fete frumoase OnlyFans")
4. Topical relevance pastrata pe fiecare site
5. Declarare transparenta in /despre/
