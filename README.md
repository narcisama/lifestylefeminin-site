# Lifestyle Feminin

Blog lifestyle feminin premium din Romania.

## Stack
- Astro 5
- TypeScript strict
- Tailwind CSS v4

## Dezvoltare locala

```bash
npm install
npm run dev
```

Acceseaza: http://localhost:4321

## Build productie

```bash
npm run build
npm run preview
```

## Deploy
Cloudflare Pages (auto-deploy din branch main)

## Documentatie
- `CLAUDE.md` - instructiuni Claude Code
- `SKILL.md` - skills active
- `docs/` - arhitectura, SEO strategy, parteneri network
